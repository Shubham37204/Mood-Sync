# 🎯 Mood-Sync

## 📌 Overview

Mood-Sync is an AI-driven solution that curates personalized entertainment experiences by dynamically recommending movies, music, and events based on user mood, preferences, and behavior.

## 🚀 Features

* Personalized recommendations
* Mood-based content filtering
* Dynamic suggestion system
* User preference tracking

## 🛠️ Tech Stack

* Frontend: React (JavaScript)
* Backend: Python (Flask)
* AI / Recommendation Logic
* APIs (for content data)
* Python Libraries:  beautifulsoup4, deepface, Flask-Cors, Flask-JWT-Extended, opencv-python, pandas, Pillow
